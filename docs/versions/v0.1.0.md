# v0.1.0: Static Analysis

**Status:** Released

## Overview

A Python CLI tool that evaluates SKILL.md files through static analysis.

## Features

- Parse SKILL.md files with YAML frontmatter
- 18 static checks across 4 dimensions (Structure, Naming, Description, Content)
- Weighted 0-100 quality score
- Console and JSON output formats

## CLI Commands

```bash
sklab evaluate ./my-skill           # Run all checks
sklab evaluate ./my-skill --spec-only  # Spec-required checks only
sklab validate ./my-skill           # Quick pass/fail validation
sklab list-checks                   # List available checks
```

## Architecture

```
src/skill_lab/
├── cli.py                    # Typer CLI
├── core/
│   ├── models.py             # Skill, CheckResult, EvaluationReport
│   ├── registry.py           # Check auto-discovery via @register_check
│   └── scoring.py            # Quality score calculation
├── parsers/
│   └── skill_parser.py       # SKILL.md parser
├── checks/
│   ├── base.py               # StaticCheck base class
│   └── static/               # 18 check implementations
├── evaluators/
│   └── static_evaluator.py   # Orchestrates check execution
└── reporters/
    ├── console_reporter.py   # Rich terminal output
    └── json_reporter.py      # JSON output
```
